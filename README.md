## For Devs:

### Create a venv with
- `python3.11 -m venv venv`
- `source venv/bin/activate`

and install venv dependencies with:

- `pip install -r requirements.txt`

### To build:
- **_Make sure you have a venv!!!_**
- run `build.sh`
